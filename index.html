Scop

Scopul aplicației este de a asigura utilizatorilor funcțiile unei telecomande pentru calculatorul personal. Astfel, utilizatorul va putea executa instrucțiuni de bază de pe propriul telefon, fără a mai fi nevoit să meargă la calculator

Funcții

  1. Control mouse
  2. Control tastatură
  3. Control funcții media
    • Sunet
    • Video
  4. Controlul calculatorului
    • Închidere
    • Restart
    • Programare închidere
  5. Funcții pentru prezentări
  6. Partajare ecran

Arhitectura

  • Aplicație Mobilă
  • Aplicație Desktop

Realizarea sistemului informatic

Fiecare instrucțiune are un cod asociat. Acest cod este transmis către aplicația desktop. În funcție de codul primit, aplicația desktop apelează o metodă din librăria user32.dll


[DllImport("user32.dll")]
        public static extern IntPtr SendMessageW(IntPtr hWnd, int Msg,
            IntPtr wParam, IntPtr lParam);

Comunicarea instrucțiunilor

  1. Internet Socket
  2. Bluetooth Socket
  3. Noi tehnologii

public interface IClient
{
    void connect();

    void sendInteger(int number);

    void sendString(String message);

    String receiveImage();

    void close();
}

Limbaje și tehnologii folosite

  • Java - Android (Fragmente, Servicii, Receptori de mesaje, Taskuri asincrone)
  • C# - Windows Forms - MetroFramework GUI - 32feet.NET - User32.dll - InstallShield
  • ASP.NET - MVC - REST API - EntityFramework - LINQ

Aplicații concurente
